# oh-my-aliases

This repository is my collection of aliases which I find useful in day to day work. Feel free to add/suggest which you find wowsome.

^The name is inspired from [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh) which is wowsome utility for zsh users.


## Installation
```
git clone https://github.com/atulsingh0/oh-my-aliases.git ~/.ohmyaliases

# Add below line in your bashrc/zshrs/kshrc file
source ~/.ohmyaliases/source_aliases.sh
```
